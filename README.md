# Specialisterne

Below are listed what week each Submodule was connected with.

## Week 1 - Introduction

This week was all introductions, so no submodules exist for this.

## Week 2 - Game Development

For week two we had to either make a Wordle or Blackjack game.
I chose Wordle, and my implementation can be found inside the [Worlde Submodule](/Specialisterne-Wordle/).

## Week 3 - Docker

For week three we got some smaller task to get familier with Docker.

With much of my time spent of game improvements, i didn't manage to do too much of the docker task.
What i did manage to do can be found inside the [Docker Submodule](/Specialisterne-Docker/).

## Week 4 - Downloader

For week four we were given a program and explanation of what it did, with the task to re-implement it with improvements to speed.

My implementation of the new Downlaoder program can be found inside the [Downloader Submodule](/Specialisterne-Downloader/).
